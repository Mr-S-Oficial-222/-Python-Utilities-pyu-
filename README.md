# PYTHON UTILITIES/ PYU

Versión: 1.0.0
Tipo: Librería / Paquete modular de código abierto
Entorno de desarrollo principal: Dispositivos móviles (Android via Pydroid 3 / Acode)
​
## Descripcion

​PYU es una librería ligera de utilidades para Python diseñada específicamente para simplificar la creación de videojuegos de texto, herramientas de consola y aplicaciones interactivas en la terminal. El proyecto nace con el objetivo de automatizar tareas repetitivas y complejas —como el formateo de colores ANSI, efectos visuales de texto y la validación segura de entradas de datos— empaquetándolas en funciones simples de una sola línea.
​Gracias a su estructura modular basada en un paquete distribuido (__init__.py y core.py), PYU ofrece una API limpia y directa que permite a desarrolladores principiantes y avanzados enfocar sus esfuerzos en la lógica pura de sus proyectos, garantizando una interfaz de usuario limpia, interactiva y robusta ante errores de ejecución.
